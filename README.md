# Paper Lens AI

Paper Lens AI is a tool designed to extract, verify, and organize author contact details from academic research papers. It uses the Groq API (Llama 3.3) to parse PDF text and DuckDuckGo to cross reference author affiliations online.

## Features

- **Batch Processing:** Upload multiple PDF files simultaneously.
- **AI Extraction:** Identifies Authors, their affiliations, and emails.
- **Web Verification:** Checks extracted data against online search results to ensure accuracy.
- **Data Export:** Download results as Excel (.xlsx) or CSV files.

## Installation


Install the required dependencies:
   pip install -r requirements.txt
